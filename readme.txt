# README do Projeto de Controle de Estoque e Sistema de Cadastro

## Descrição

Este projeto consiste em dois sistemas desenvolvidos em Excel utilizando VBA (Visual Basic for Applications): um sistema de controle de estoque e um sistema de cadastro de usuários. O objetivo é facilitar a gestão de produtos e informações de clientes, permitindo um registro e consulta eficientes.

## Projetos Incluídos

### 1. Controle de Estoque

#### Descrição
O sistema de controle de estoque permite gerenciar produtos, acompanhar entradas e saídas de itens, e gerar relatórios sobre o status do estoque.

#### Funcionalidades
- **Cadastro de Produtos**: Adicionar novos produtos ao estoque.
- **Controle de Entradas e Saídas**: Registrar movimentações de produtos.
- **Relatórios**: Gerar relatórios de estoque atual e histórico de movimentações.

#### Estrutura do Projeto

MultipleFiles/ 
│ 
├── Controle de Estoque -  # Arquivo principal do sistema de controle de estoque


#### Como Executar o Projeto
1. **Pré-requisitos**:
   - Microsoft Excel instalado em sua máquina.

2. **Acessando o Arquivo**:
   - Navegue até a pasta `MultipleFiles` onde o arquivo está armazenado.

3. **Abrindo o Arquivo**:
   - Abra o arquivo `Controle de Estoque.xlsm` para utilizar o sistema de controle de estoque.

---

### 2. Sistema de Cadastro

#### Descrição
O sistema de cadastro permite gerenciar informações de clientes ou usuários, facilitando o registro e a consulta de dados.

#### Funcionalidades
- **Cadastro de Usuários**: Adicionar novos usuários ao sistema.
- **Edição de Dados**: Atualizar informações de usuários existentes.
- **Consulta de Usuários**: Pesquisar e visualizar dados de usuários cadastrados.
